# YouTube Clone App

Cross-platform YouTube clone mobile app built with React Native for iOS and Android. Demonstrates core mobile development skills including UI implementation, navigation patterns, video playback integration, and cross-platform compatibility.

## Features
- YouTube-inspired UI layout with dark theme
- Bottom tab navigation (Home, Trending, Subscriptions, Library)
- Video player component integration
- Search bar and content discovery
- Responsive design across different screen sizes
- Smooth animations and transitions

## Tech Stack
- React Native (CLI bootstrapped)
- JavaScript (84.9%) + TypeScript (10.6%)
- Metro bundler for development
- Native iOS/Android builds via Xcode/Android Studio

## Development Workflow
- Edit `App.tsx` for main app changes
- Android reload: Press `R` twice or shake device
- iOS reload: Cmd + R in Simulator
- Developer menu: Ctrl + M (Windows/Linux) or Cmd + M (Mac)
- Live reload and hot reloading enabled
